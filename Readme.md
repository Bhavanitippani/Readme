# Project Explanation — Online Food Delivery Analytics

## Business Problem:
Modern food delivery platforms generate large volumes of transactional and behavioural customer data.
Analyse order patterns, revenue, customer behaviour and product performance.
The goal of this project is to extract insights that can help improve business decisions such as:

## Dataset Description:
The project uses a structured relational dataset consisting of:
- customers.csv → customer information
- orders.csv → order-level data
- items.csv → food items
- payments.csv → payment data

## SQL Analysis Performed:
Using SQL, several key analytical queries were performed:
- customer retention queries
- revenue by category
- time-based sales trend
- top purchased items
- total revenue, avg order value

## PowerBI Dashboard:
A visually interactive dashboard was created to clearly present insights including:
- Total Sales
- Orders by hour
- Customer segmentation
- Item level performance
- Payment distribution

## Outcome:
Evening time shows maximum ordering activity, indicating peak marketing window
- Found evening peak orders
- Identified high-value customer segment
- Recommended strategic promo timing
Strategic promo timing and targeted offers can improve sales
